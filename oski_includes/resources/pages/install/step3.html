    This is the last time we're going to ask you for information during this install.

    What we need are Database credentials.

    Now, the OSKI ENGINE officialy supports both MySQL and MongoDB. Choose one from the list and supply the data required.

    What does the Prefix mean? It means that instead of the table (or collection) named 'users' the table will be 'module_users'. This is particulary usefull if you want to use the database for something other than module engine.

    The Admin Account is created anyway, with the username 'admin' and the password '123456'. You are given now the opportunity to modify these from the start!
